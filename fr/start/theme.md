# Application d'un thème

Nous en parlions, il est temps de donner une apparence un peu personalisée à votre travail.

Vous pouvez déjà vous inspirer d'un certains nombre de [template](http://getbootstrap.com/getting-started/#examples). Au passage, vous verrez qu'ils ne vous sont pas inconnus, car utilisés souvent tels quels…

Du coup, pour commencer, nous allons nous approprier un thême déjà existant sur un des nombreux sites. http://bootswatch.com propose par exemple des travaux gratuits.

[code](http://jsbin.com/dayule/2/embed?output)

Nous avons donc remplacer le la feuille de style par celle de bootswatch. Elle contient des valeurs identiques aux règles de Bootstrap, mais avec un niveau de personnalisation intéressant.

> **Hint** Entrainez-vous à modifier les valeurs de styme directement dans l'inspecteur de code de votre navigateur.

![Personnalisation avec l'inspecteur web](../images/webinspector.png)

## Bonnes pratiques pour le redéfinition des styles :

- Evitez de modifier les sources directement
- Attention à l'ordre de déclaration des styles
- Ré-iutilisez les sélecteurs natifs
- Evitez de surcharger les priorités des règles

> **Hint** L'usage de LESS nous permettra de configurer nous-même les styles employés.
